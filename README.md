# Linear Data Structures Toolkit (LDST)

## Overview
This project is part of Unit–2 (Linear Data Structures) assignment.  
It implements fundamental data structures from scratch using Python and demonstrates their real-world application.

## Features
- Dynamic Array implementation with resizing
- Singly Linked List (insert, delete, traverse)
- Doubly Linked List (insert after node, delete at position)
- Stack (LIFO) using linked list
- Queue (FIFO) using linked list
- Balanced Parentheses Checker using Stack

## Files Included
- `ldst_toolkit.py` → Main Python code
- `report.pdf` → Explanation and analysis
- `output.txt` → Program output

## How to Run
Run the following command in terminal:
python ldst_toolkit.py
